# NideShop 后台管理 Demo

这是一个独立静态后台页面，入口文件：

```text
admin-html/index.html
```

## 使用方式

直接用浏览器打开 `index.html` 即可。

登录页有两种模式：

- API 模式：连接 `http://127.0.0.1:8360/admin/`
- 离线演示：不需要后端和数据库，使用页面内置演示数据

## 已实现页面

- 数据概览
- 商品管理：列表、搜索、新增、编辑、删除
- 活动管理：列表、搜索、新增、编辑、删除
- 用户管理：列表、搜索、查看
- 订单管理：列表、搜索、查看
- 品牌管理：列表、搜索、新增、编辑、删除
- 分类管理：列表、新增、编辑、删除

## 对接后端

后端项目目录：

```text
nideshop-master/nideshop-master
```

后端启动后，后台页面默认请求：

```text
http://127.0.0.1:8360/admin/
```

请求会携带：

```text
X-Nideshop-Token: <登录返回 token>
```

商品新增需要商品表可写。当前已对后端商品新增逻辑做了一个小兼容：如果新增时没有传 `id`，会取当前最大商品 ID 加 1。
